# GSE16131 - Follicular Lymphoma Gene Expression Dataset

## Overview
- **GEO Accession**: GSE16131
- **Title**: Differences Between Follicular Lymphoma With and Without Translocation t(14;18)
- **Samples**: 184 FL patients (grade 1-3A)
- **Platform**: Affymetrix Human Genome U133A (GPL96) and U133B (GPL97) Arrays
- **Publication Date**: June 2009
- **PubMed ID**: 19471018

## Clinical Data Available
- Follow-up status (alive/dead)
- Follow-up time (years)
- t(14;18) translocation status
- Individual patient IDs

## Files
- `GSE16131-GPL96_series_matrix.txt.gz` - Expression data + clinical annotations (U133A array)
- `GSE16131-GPL97_series_matrix.txt.gz` - Expression data + clinical annotations (U133B array)
- `GSE16131_family.soft.gz` - Full SOFT format file with all metadata

## Data Processing Notes
- Fresh-frozen tumor biopsies from untreated FL patients
- Pre-rituximab era cohort (diagnosed 1974-2001)
- Commonly used as training set for survival prediction models

## Key Publications
- Dave et al. (2004) NEJM - Original gene expression profiling study
- Leich et al. (2009) - t(14;18) subgroup analysis (this dataset)
