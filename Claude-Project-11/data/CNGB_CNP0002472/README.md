# CNGB CNP0002472 - FL Histologic Grades WES Dataset

## Overview
- **Accession**: CNP0002472
- **DOI**: 10.26036/CNP0002472
- **Title**: Genetics and Pathogenesis of Follicular Lymphoma (WES)
- **Samples**: 200 samples (149 with WES data)
- **Total Size**: 1.35 TB
- **Data Type**: Whole Exome Sequencing (WES)

## Study Details
This dataset is part of the study "Molecular landscape of distinct follicular lymphoma histologic grades: insights from genomic and transcriptome analyses" (Leukemia 2025).

### Clinical Cohort
- 831 patients with clinical data
- 149 patients with WES
- FL grades: FL1-2, FL3A, FL3B, FL/DLBCL

### Key Findings
- FL1-2 and FL3A share common genetic background (indolent)
- FL3B and FL/DLBCL lack CREBBP/KMT2D mutations (aggressive)
- Grade-specific mutation patterns identified
