# Controlled-Access Datasets for Follicular Lymphoma

These datasets require application through EGA or dbGaP.

---

## dbGaP Datasets (US - NIH)

### phs002989 - Mayo Clinic FL Multi-omics (BEST OPTION)
- **Description**: FL B cell transcriptional states with matched WES
- **Samples**: 87 RNA-seq + 123 WES (matched)
- **Data Types**: Bulk RNA-seq, WES, immune profiling
- **Outcomes**: Transformation risk, survival associations
- **Publication**: [Krull et al. Cell Reports Medicine 2024](https://pmc.ncbi.nlm.nih.gov/articles/PMC10983045/)
- **URL**: https://www.ncbi.nlm.nih.gov/gap/ (search phs002989)

### phs001229 - Alliance A151303 POD24
- **Description**: Mutation analysis for progression prediction
- **Samples**: 370 FL patients
- **Data Types**: Targeted mutation panel (7 key genes)
- **Outcomes**: POD24, PFS, transformation
- **Publication**: [Blood Advances 2023](https://ashpublications.org/bloodadvances/article/7/18/5524/497092/)
- **URL**: https://www.ncbi.nlm.nih.gov/gap/ (search phs001229)

### phs001378 - Single-cell RNA-seq FL
- **Description**: scRNA-seq of FL tumors
- **Samples**: 6 FL tumors (~34,000 cells)
- **Data Types**: 10X scRNA-seq
- **Publication**: [Blood 2019](https://ashpublications.org/blood/article/133/10/1119/272746/)
- **URL**: https://www.ncbi.nlm.nih.gov/gap/ (search phs001378)

### How to Apply (dbGaP)
1. Go to https://www.ncbi.nlm.nih.gov/gap/
2. Search for the accession (e.g., phs002989)
3. Click "Request Access"
4. Requires eRA Commons account (institutional)
5. Submit Data Access Request (DAR)
6. Approval typically 2-4 weeks

## EGA Datasets (Europe)

### EGAS00001000399 - FL Transformation WGS
- **Samples**: 6 patients, 20 genomes (FL-tFL pairs)
- **Data Type**: WGS
- **URL**: https://www.ega-archive.org/studies/EGAS00001000399

### EGAD00001004109 - FL RNA-seq
- **Samples**: 28 FL samples
- **Data Type**: RNA-seq
- **URL**: https://ega-archive.org/datasets/EGAD00001004109

### EGAD00001001301 - FL WES (Untransformed)
- **Data Type**: WES
- **URL**: https://ega-archive.org/datasets/EGAD00001001301

### How to Apply (EGA)
1. Create account at https://ega-archive.org/
2. Navigate to dataset page
3. Click "Request Access"
4. Submit to Data Access Committee (DAC)
5. Approval typically 2-4 weeks
