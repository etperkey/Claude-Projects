# KanLab - Demo Projects

This folder contains documentation for demo projects showcasing KanLab functionality.

## Projects

1. **Machine Learning Pipeline** (`ml-pipeline.md`) - Data science infrastructure project
2. **API Integration Platform** (`api-platform.md`) - Microservices architecture project
3. **User Analytics Dashboard** (`analytics-dashboard.md`) - Business intelligence project
4. **Mobile App Development** (`mobile-app.md`) - Cross-platform solution project

## Using KanLab

This demo showcases the following features:

- **Kanban Boards**: Drag-and-drop task management
- **Lab Notebooks**: Project-specific documentation
- **Literature Manager**: Reference tracking with citations
- **Calendar View**: Timeline and deadline management
- **Global Search**: Search across all projects and content
- **AI Integration**: Support for Claude, OpenAI, and Gemini APIs
- **Google Sync**: Calendar and Drive integration
- **Export/Import**: Data portability features

## Getting Started

1. Navigate to a project from the landing page
2. Use the Kanban board to manage tasks
3. Add notes to the Lab Notebook
4. Track references in the Literature Manager
5. Use the calendar to view deadlines
